# kratosNet — Οδηγός Εγκατάστασης (για τον Κωνσταντίνο)

Όλα τα βήματα με τη σειρά, από το μηδέν μέχρι live εφαρμογή + Play Store.
Χρόνος: ~30-40 λεπτά ενεργής δουλειάς (χωρίς τους χρόνους αναμονής approval).

**Τι ΔΕΝ χρειάζεται να κάνεις**: κανένα build, κανένα τοπικό dev setup, καμία
πληρωμή (εκτός από το εφάπαξ $25 του Play Store, μόνο αν θες Android app).

---

## ΦΑΣΗ 1 — Live στο internet (10 λεπτά) ✅ υποχρεωτική

### 1.1 GitHub
```bash
cd govhub          # ο φάκελος από το kratosnet.zip
git init
git add .
git commit -m "kratosNet v1"
gh repo create kkitsop/kratosnet --private --source=. --push
```
Χωρίς `gh` CLI: φτιάξε repo στο github.com, μετά:
```bash
git remote add origin https://github.com/kkitsop/kratosnet.git
git branch -M main && git push -u origin main
```

### 1.2 Cloudflare Pages
1. dash.cloudflare.com → **Workers & Pages → Create → Pages → Connect to Git**
2. Διάλεξε το `kratosnet` repo
3. Framework preset: **None** · Build command: **(κενό)** · Output dir: **/**
4. **Save and Deploy** → σε ~1' έχεις `https://kratosnet-XXX.pages.dev`

### 1.3 Πρώτος έλεγχος
Άνοιξε το URL στο **κινητό σου** (όχι στο preview panel!) και τσέκαρε:
- [ ] Υπηρεσίες: πατάς π.χ. Διαύγεια → ανοίγει ΜΕΣΑ στην εφαρμογή, με 🔒 domain
- [ ] TAXISnet → ανοίγει σε νέα καρτέλα browser (σωστό, είναι σκόπιμο)
- [ ] ΕΣΠΑ/ΚΑΔ: γράψε 56.30 → βγάζει το πρόγραμμα Δυτικής Ελλάδας
- [ ] Κοντά μου → ATM: ζητάει τοποθεσία, δείχνει κοντινά με αποστάσεις
- [ ] Δήμος: φορτώνει αναρτήσεις Διαύγειας (αν δείχνει fallback, δες §5 παρακάτω)
- [ ] Το ? πάνω αριστερά ανοίγει τη Βοήθεια/FAQ
- [ ] 🌓 αλλάζει θέμα
- [ ] Προσθήκη στην αρχική οθόνη → ανοίγει σαν κανονικό app

### 1.4 Ενεργοποίηση nightly scraper
GitHub repo → **Actions** → αν ρωτήσει, "I understand… enable workflows" →
**Nightly ESPA scrape → Run workflow** (χειροκίνητο πρώτο τεστ).
Μετά τρέχει μόνο του κάθε βράδυ 06:15 ώρα Ελλάδας.

**Έλεγχος logs στο πρώτο run**: πρέπει να δεις `[ok] σελίδα 1: σύνολο ...` και
αριθμό να ανεβαίνει σελίδα-σελίδα. Αν κολλήσει στη σελίδα 1 με «Τέλος
σελιδοποίησης», στείλε μου τα logs — γνωστό πιθανό σημείο, φτιάχνεται με μικρή
αλλαγή selector (εξηγείται στο README).

## ΦΑΣΗ 2 — Λογαριασμοί + Email ειδοποιήσεις (15 λεπτά) 🔶 προαιρετική αλλά προτεινόμενη

### 2.1 Supabase (δωρεάν, χωρίς κάρτα)
1. supabase.com → New project (Free) → περιμένεις ~2' να στηθεί
2. **SQL Editor → New query** → επικόλλησε ΟΛΟ το περιεχόμενο του
   `supabase/schema.sql` → **Run** (πρέπει να πει Success)
3. **Settings → API**: κράτα ανοιχτά τα **Project URL** και **anon public** key
4. **Authentication → URL Configuration → Site URL**: βάλε το pages.dev URL σου

### 2.2 Σύνδεση frontend
Άνοιξε το `index.html`, βρες (Ctrl+F) το `SUPABASE_CONFIG` και συμπλήρωσε:
```js
const SUPABASE_CONFIG = {
  url: "https://XXXX.supabase.co",   // το Project URL
  anonKey: "eyJhbGciOi...",           // το anon public key
};
```
Commit + push → το Cloudflare κάνει redeploy μόνο του σε ~1'.
⚠️ Το **anon** key. ΠΟΤΕ το service_role στο index.html.

### 2.3 Brevo (δωρεάν, χωρίς κάρτα — για τα emails ειδοποιήσεων)
1. brevo.com → Sign up → επιβεβαίωσε το email σου
2. **Settings → Senders** → πρόσθεσε/επιβεβαίωσε το email αποστολέα (το Gmail σου κάνει)
3. **SMTP & API → API Keys → Generate a new API key** → κράτα το

### 2.4 GitHub Secrets
Repo → **Settings → Secrets and variables → Actions → New repository secret** (×5):

| Name | Value |
|---|---|
| `SUPABASE_URL` | το Project URL |
| `SUPABASE_SERVICE_KEY` | Supabase → Settings → API → **service_role** |
| `BREVO_API_KEY` | από το 2.3 |
| `NOTIFY_SENDER_EMAIL` | το verified sender email |
| `APP_URL` | το pages.dev URL |

### 2.5 End-to-end τεστ (σημαντικό!)
1. Στο app (κινητό): ΕΣΠΑ tab → Λογαριασμός → βάλε το email σου → σου έρχεται
   magic link → πάτα τον → συνδέεσαι
2. Βάλε ΚΑΔ 56.30, ενεργοποίησε τον διακόπτη ειδοποιήσεων ("Συγχρονίστηκε ✓")
3. GitHub → Actions → Run workflow χειροκίνητα → δες στα logs:
   `[ok] 1 χρήστες με ενεργές ειδοποιήσεις` και `[sent] ...`
4. Έλεγξε το inbox σου (και τα spam την πρώτη φορά)

*Ειλικρινής σημείωση: αυτή τη ροή δεν μπόρεσα να τη δοκιμάσω end-to-end από το
dev περιβάλλον (μόνο τη λογική με mock δεδομένα, 4/4 σενάρια σωστά). Αν κάτι
στραβώσει στο βήμα 3, τα logs θα δείχνουν ακριβώς πού — στείλ' τα μου.*

## ΦΑΣΗ 3 — Android / Play Store (δικός σου χρόνος + $25) 🔷 όποτε θες

Προϋπόθεση: η Φάση 1 ολοκληρωμένη (χρειάζεται το live HTTPS URL).

1. **pwabuilder.com** → βάλε το URL σου → Package for stores → **Android**
   - Package ID: `gr.kratosnet.twa` (ή ό,τι προτιμάς)
   - Signing: "Create new" (το PWABuilder κρατάει το key — κατέβασέ το και ΦΥΛΑΞΕ το)
   - Κατεβάζεις το **.aab** + σου δείχνει ένα **SHA256 fingerprint**
2. Άνοιξε `.well-known/assetlinks.json` στο repo → αντικατέστησε το
   `REPLACE_WITH_SHA256_...` με το πραγματικό fingerprint → commit + push
3. **play.google.com/console** → πληρωμή $25 (εφάπαξ) → Create app → ανέβασε το .aab
4. Θα σου ζητήσει:
   - Screenshots: τράβα από το κινητό σου (τουλάχιστον 2)
   - Εικονίδιο 512×512: υπάρχει ήδη (`icon-512.png`)
   - Privacy Policy URL: **υποχρεωτικό** — πες μου και σου γράφω τη σελίδα σε 5'
   - Περιγραφή, κατηγορία (Productivity), content rating ερωτηματολόγιο
5. Review Google: 1-7 μέρες συνήθως

⚠️ **Πριν το υποβάλεις**: στο store listing (τίτλος/περιγραφή/γραφικά) βάλε ξεκάθαρα
«Ανεξάρτητη εφαρμογή — δεν σχετίζεται με το Ελληνικό Δημόσιο». Το app έχει ήδη το
disclaimer μέσα στο Help/FAQ, αλλά το listing κρίνεται χωριστά και η πολιτική Google
για apps που θυμίζουν κυβερνητικές υπηρεσίες είναι αυστηρή.

## Γρήγορη αναφορά αρχείων

| Αρχείο | Τι είναι |
|---|---|
| `index.html` | Ολόκληρη η εφαρμογή (single file) |
| `sw.js` | Offline support |
| `manifest.json` | PWA config + shortcuts + icons |
| `data/programs.json` | Δεδομένα ΕΣΠΑ (ανανεώνονται νυχτερινά) |
| `scraper/espa_scraper.py` | Nightly scraper (Playwright) |
| `scraper/notify_kad.py` | Nightly email ειδοποιήσεις |
| `supabase/schema.sql` | Database schema (τρέχει 1 φορά) |
| `.well-known/assetlinks.json` | Android domain verification |
| `.github/workflows/scrape-espa.yml` | Το nightly automation |
| `README.md` | Πλήρης τεχνική τεκμηρίωση + ιστορικό αλλαγών |

## Αν κάτι πάει στραβά — πρώτα βήματα
- **Scraper κολλάει σε 10 προγράμματα** → README, ενότητα "Γνωστός περιορισμός" → στείλε logs
- **Δήμος tab μόνιμα σε fallback** → browser console → Network tab → δες τι επιστρέφει το diavgeia.gov.gr request → στείλ' το μου
- **Magic link δεν έρχεται** → Supabase έχει όριο ~4 auth emails/ώρα στο free — περίμενε ή δες Auth → Logs
- **Emails ειδοποιήσεων δεν φεύγουν** → Actions logs του "Send KAD notifications" step λένε ακριβώς τι λείπει

Καλή επιτυχία — και καληνύχτα! 🌙
